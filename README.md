# Online Marketplace Database

## Быстрый старт

### Требования
- [Docker](https://docs.docker.com/get-docker/) и Docker Compose

### Запуск

```bash
# Запустить всё (PostgreSQL + Web UI)
make up
```

После запуска:
- **Web UI**: [http://localhost:8080](http://localhost:8080)
- **psql консоль**: `make psql`

### Все команды

```bash
make help                                    # Справка по всем командам
make up                                      # Запустить PostgreSQL + Web UI
make psql                                    # Открыть интерактивную консоль psql
make queries                                 # Выполнить все 50 запросов
make query SQL="SELECT * FROM products"      # Выполнить один запрос
make init                                    # Пересоздать схему и перезагрузить данные
make down                                    # Остановить контейнеры
make clean                                   # Остановить и удалить все данные
make logs                                    # Логи PostgreSQL
make web-logs                                # Логи Web UI
```

## Как это работает

1. **Первый запуск** (`make up`): Docker создаёт volume `pgdata`, PostgreSQL выполняет скрипты из `init/` — создаёт 17 таблиц и заполняет тестовыми данными. Это происходит **один раз**.

2. **Последующие запуски**: PostgreSQL видит, что volume уже содержит данные, и **пропускает** инициализацию. Контейнер стартует с готовой базой за ~3 секунды.

3. **Сброс данных**: `make clean` удаляет volume, после чего следующий `make up` пересоздаст всё заново.
